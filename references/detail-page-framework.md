# 详情页分析与提示词框架

## 使用范围

在第一阶段分析竞品详情页、第二阶段编排详情页逐屏任务时读取本文件。只学习竞品的页面角色、证据方式、版式和叙事顺序，不复制无法核验的功效、数字或背书。

## 共通叙事框架

按以下顺序组织详情页：

1. 信任建立或主利益：品牌、真实资质、核心购买理由；
2. 痛点激活：目标人群在具体场景遇到的麻烦；
3. 核心方案：产品通过什么事实或结构回应问题；
4. 机理证明：局部示意、结构拆解、受力/作用路径；
5. 材质与细节：材质特写、工艺、尺寸、结构差异；
6. 人群与场景：谁在什么情况下使用，如何代入；
7. 选择决策：SKU、尺码、组合、包装、配件；
8. 信息收口：参数、资质、方法、注意事项、适用边界。

不要从参数堆砌开始。前段回答“为什么继续看”，中段回答“为什么相信”，后段回答“怎么选、怎么买、怎么用”。

## 竞品单屏总结字段

逐屏记录以下字段：

| 字段 | 要求 |
| --- | --- |
| 页码 | 使用素材真实编号 |
| 页面角色 | 信任、痛点、核心卖点、机理、材质、场景、选择、参数、注意事项 |
| 可见主标题 | 只抄录清晰可见文字；看不清写“需复核” |
| 产品事实 | 页面依赖的可核验事实 |
| 消费者问题 | 回应的具体麻烦 |
| 消费者结果 | 用户可感知的结果 |
| 视觉证据 | 产品图、特写、对比、箭头、场景、证书、参数卡等 |
| 可学习结构 | 可复用的叙事或版式 |
| 合规风险 | 医疗化、绝对化、前后对比、无证据数据、外部背书等 |

汇总时回答：前 3 屏如何留住用户、中段如何证明差异、后段如何降低选择成本、哪些结构可复用、哪些表达必须改写或删除。

## 页面地图

普通个护/日百通常规划 8-12 屏：

1. 核心利益/品牌信任；
2. 用户痛点；
3. 核心解决方案；
4. 结构或材质机理；
5. 细节特写；
6. 使用场景；
7. SKU/规格/选择；
8. 使用方法；
9. 参数/标准/注意事项；
10. 包装/服务/信任收口。

医疗器械或高风险类目通常规划 10-14 屏：

1. 品牌/资质信任；
2. 使用问题或场景，不夸大病症；
3. 产品核心结构；
4. 按说明书范围表达的作用机理；
5. 材质和工艺；
6. 支撑、固定或保护路径；
7. 佩戴/使用步骤；
8. 场景和人群；
9. 尺码/SKU；
10. 产品参数；
11. 注意事项和适用边界；
12. 注册证/备案信息信任收口，仅限医疗器械且用户确认需要展示。

医疗器械详情页如展示注册证/备案证明，应上传证照原图作为参考图，用画框/展示框承载证照，并在提示词中严格限定证照文字、编号、印章、表格和版式保持参考图原样；可在证照外围增加背景、标题和辅助说明。证照缩放后如不可读，可要求画面加入“局部放大框”，但放大框内容仍必须来自上传参考图，不得让模型仿写证照文字。非医疗器械产品不固定规划注册证或备案证明页。

页数按素材证据和产品复杂度调整，不为凑页数重复卖点。

## 连贯性规则

先建立整套视觉母版：

- 固定品牌主色和类目辅助色；
- 固定大标题、副标题、卖点卡、参数小字的层级；
- 固定图标、信息卡、箭头和背景风格；
- 每 2-3 屏让产品包装或产品主体重新出现；
- 参数卡、资质卡、步骤卡保持同一组件样式。

固定叙事主线：

`信任建立 → 真实痛点 → 核心方案 → 为什么能做到 → 细节证明 → 怎么用/谁适合 → 怎么选 → 参数资质收口`

每屏只承担一个说服任务。页面地图和展示版表格可以记录“承接上一屏”，但发给图像 API 的 `生成提示词` 不写前后承接关系。不要在后段重新引入无关新痛点。普通商品最后回到规格、方法、日常使用价值和注意事项；医疗器械如用户确认需要证照页，可在完成规格与注意事项后，用上传证照参考图的画框展示作为信任收口。

## 单屏提示词结构

每条发给图像 API 的提示词只写当前屏需要的内容：

```text
参考图：{说明已上传参考图如何使用}
画面规格：{目标尺寸/比例}
画面描述：{产品/局部特写/人体示意/场景/证书/参数卡，结构拆解/材质微距/箭头/步骤/选择卡等}
页面中需要带出的文字：
主标题：{...}
副标题：{...}
辅助信息：{...}
底部条：{...}
```

把通用禁用项、合规限制、复核提醒放到复核清单，不写进提示词主体；不要在提示词里写“不要出现”“不得生成”“避免”等负向控制语。

## 类目差异

- 口腔护理：强调清洁不到位等具体问题，以牙缝微距、线体/贴片结构、使用动作、材质和标准作证据；谨慎处理美白、去黄、效果对比。
- 护腰/医疗器械：强调品牌/资质、人体工学结构、支撑路径、面料、佩戴步骤、尺码和注册信息；所有用途和功效必须服从我品注册证、说明书和标签。

## 生成后复核

用总览图检查：色系和组件是否统一、产品是否一致、页面顺序是否自然、转场是否成立、文案是否可读、是否混入竞品品牌/平台/二维码/联系方式，以及后段是否完整包含选择与合规信息。
